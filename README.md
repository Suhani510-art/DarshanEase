DarshanEase – Temple Darshan Ticket Booking App

 Overview
DarshanEase is a web application that allows devotees to book temple darshan slots online. It helps users avoid long queues by allowing them to check available time slots and book their visit in advance.

 Features
- User Login and Registration
- View temples and their details
- Check available darshan slots
- Book darshan tickets online

 Tech Stack
- Frontend: React.js, CSS, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB

 Purpose
The main goal of this project is to make temple darshan booking easier and more convenient for devotees by using an online platform.
